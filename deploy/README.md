# Getting Started


## Build and Deploy Proposals

### 1. Register `AXL` in vbank Proposal

This is required for the contract to function as expected.

```sh
yarn build:token
# assumes agoric-to-axelar-local is running
yarn deploy:token
```

### 2. Yield Max Contract Proposal

```sh
yarn build
# assumes agoric-to-axelar-local is running
yarn deploy
```
